# CLAUDE.md
Проект: Nadira
Описание
Мобильное приложение Nadira для платформ Android и iOS.
Публикация в Google Play Market и Apple App Store.
Технологии

Язык: TypeScript (React Native)
Фреймворк: React Native с Expo
Платформы: Android и iOS

Правила написания кода

Все комментарии писать на русском языке
Все переменные и функции называть на английском языке
Код должен работать на обеих платформах: Android и iOS
Использовать TypeScript (не чистый JavaScript)
Компоненты разбивать на маленькие переиспользуемые части

Структура проекта

/src/screens — экраны приложения
/src/components — переиспользуемые компоненты
/src/navigation — навигация между экранами
/src/services — работа с API и данными
/src/assets — картинки, шрифты, иконки

Важные правила

Всегда проверять что код работает на Android И на iOS
Не использовать библиотеки которые работают только на одной платформе
Перед публикацией проверять требования Google Play и App Store
Писать понятный код с комментариями на русском

Цель
Создать качественное мобильное приложение которое можно легко расширять и добавлять новые функции в будущем.
